Description --
----------------------------------------------------

Used Python to create a Desktop Application that can fill colors to a gray-scaled image, including pencil sketches.



Instructions --
----------------------------------------------------

1.  Download and extract the repository.

2.  Inside “Chromatic-Pen/static/model” folder, extract “model.zip.001” using 7-zip.

3.  Inside “Chromatic-Pen” folder, extract “main.zip.001” using 7-zip.

4.  Execute “main.exe”.
